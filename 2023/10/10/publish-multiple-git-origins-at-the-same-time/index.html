同时推送多个 Git 仓库提交

在使用 Git 时，有一种场景是需要将代码同时推送到多个仓库中，分别进行提交、推送也勉强可以，但是太不优雅了！经过简单的资料查阅，我发现 Git 本身其实提供了这样的命令来完成这项任务。

在多方对比后，我选择了 “一次推送” 的方案。直接修改了项目本地配置文件 .git/config，在 remote "origin" 做了如下设置：
